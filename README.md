# Six Shades of Nature Scene in OpenGL

This project is a C++ and OpenGL based 2D animated village environment that simulates different shades of nature through animation.

# Features :
1. Day and night transition
2. Dynamic weather (rain and snowfall)
3. Moving boat
4. Village elements like houses, trees, river, sun, moon, stars and clouds

# Technology Used :
1. C++
2. OpenGL (GLUT)
